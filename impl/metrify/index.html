metrify/config.py

Exports configuration data for the Flask application

class metrify.config.Config

Configuration class for the Flask application.

Note:

Should be used with from_object

Example:
>>> config = Config
>>> app.config.from_object(config)
Variáveis:

  • MONGO_URI (str) – URI for the MongoDB connection

  • SCHEDULER_API_ENABLED (bool) – APScheduler-enabled API integration
