hello – Módulo de Demonstração e Exemplificação Técnica

metrify/hello/__init__.py

The hello package serves the sole purpose of demonstrating good practices and design patterns that should be enforced in the project’s code and structuring.

This module exports a Blueprint object with a single registered route, and contains an “strategies.py” file, which serves as a “backend” for the exposed endpoint.
